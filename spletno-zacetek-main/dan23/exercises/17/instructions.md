# Vaja 17: Lokalna shramba (osnove)

## Cilji
- Spoznati localStorage API
- Shraniti podatke
- Pridobiti shranjene podatke
- Prikazati shranjene podatke

## Navodila
1. Shranite uporabniško ime:
   - Vnosno polje za ime
   - Gumb za shranjevanje
   - localStorage.setItem()

2. Pridobite podatke ob ponovnem nalaganju:
   - window.onload dogodek
   - localStorage.getItem()
   - Obdelava null vrednosti

3. Prikažite shranjene podatke:
   - Element za prikaz
   - Dinamična posodobitev
   - Obdelava neobstoječih podatkov

4. Dodajte funkcionalnost brisanja:
   - Gumb za brisanje
   - localStorage.removeItem()
   - Posodobitev prikaza

## Pričakovani rezultat v script.js
- Vnosno polje in gumb za shranjevanje
- Gumb za brisanje
- Funkcije za shranjevanje in pridobivanje
- Element za prikaz shranjenih podatkov

## Namigi
- Preverite, da deluje po osvežitvi strani
- Poskusite shraniti različne podatke
- Preverite, da je brisanje učinkovito
