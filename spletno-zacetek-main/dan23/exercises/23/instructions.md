# Vaja 23: Razredi

## Cilji
- Spoznati JavaScript razrede
- Uporabiti konstruktor
- Definirati metode
- Instancirati objekte
- Razumeti dedovanje

## Navodila
1. Uporabite razredno sintakso:
   - class deklaracija
   - constructor metoda
   - method definicije

2. Uporabite konstruktor:
   - Inicializacija lastnosti
   - Parametri konstruktorja
   - this referenca

3. Definirajte metode:
   - Instance metode
   - Static metode
   - Getter in setter metode

4. Instancirajte objekte:
   - new operator
   - Parametri pri instanciranju
   - Dostop do lastnosti in metod

5. Razumite dedovanje:
   - extends ključna beseda
   - super() klic
   - Prekrivanje metod

## Pričakovani rezultat v script.js
- Razredna deklaracija
- Konstruktor z lastnostmi
- Metode razreda
