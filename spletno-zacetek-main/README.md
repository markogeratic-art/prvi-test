# Osnove spletnega programiranja

**Tehnologije:** HTML, CSS, JavaScript, TypeScript, FastAPI, jQuery, AJAX  
**Trajanje:** 5 dni približno 30 ur  
**Obseg na dan:** približno 4 ure in 30 minut  
**Cilj sklopa:** Razumevanje delovanja spletnih strani in osnovnih spletnih tehnologij ter povezave med frontend in backend delom
