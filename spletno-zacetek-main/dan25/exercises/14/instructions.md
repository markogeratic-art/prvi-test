# Vaja 14: Izbirnik barv

## Cilji
- Uporabiti color input element
- Implementirati živo posodabljanje
- Uporabiti dogodke input
- Spremeniti barvo ozadja

## Navodila
1. Ustvarite color input:
   - <input type="color"> element
   - Privzeta barva
   - Spremenljivka za shranjevanje izbire

2. Uporabite dogodek input:
   - Poslušalec dogodka input
   - Reakcija na vsako spremembo
   - Pridobitev trenutne barve

3. Implementirajte živo posodabljanje:
   - Spremenite barvo ozadja
   - Uporabite style.backgroundColor
   - Posodobitev brez osveževanja strani

4. Dodajte vizualni feedback:
   - Prikažite izbrano barvo
   - Dodajte besedilo z hex kodo
   - Posodobitev v realnem času

## Pričakovani rezultat v script.ts
- Color input element
- Poslušalec dogodka input
- Funkcija za posodabljanje barve
- Element za prikaz izbrane barve
- TypeScript type assertion za input element
