# Vaja 08: Objekti

## Cilji
- Ustvariti in upravljati objekte
- Dostopati do lastnosti različnimi načini
- Dodajati in odstranjevati lastnosti
- Uporabiti metode objektov
- Zanka čez objekte

## Navodila
1. Ustvarite objekte:
   - Literalski način
   - S pomočjo konstruktorja
   - S pomočjo Object.create()

2. Dostopajte do lastnosti:
   - Z piko (object.property)
   - Z oglatimi oklepaji (object["property"])
   - Dinamično z spremenljivkami

3. Dodajajte in odstranjevajte lastnosti:
   - Dodajte nove lastnosti
   - Odstranite obstoječe lastnosti
   - Posodobitev vrednosti lastnosti

4. Uporabite metode objektov:
   - Definirajte metode znotraj objekta
   - Uporabite this referenco
   - Pokličite metode

5. Zanka čez objekte:
   - for...in zanka
   - Object.keys() metoda
   - Object.values() metoda

## Pričakovani rezultat v script.ts
- Primeri ustvarjanja objektov
- Dostop do lastnosti različnimi načini
- Dodajanje in odstranjevanje lastnosti
- Primeri metod objektov
- Zanka čez objekte
- TypeScript interface za objekte

## Namigi
- Preverite, da so lastnosti pravilno dostopane
- Poskusite kombinirati različne načine ustvarjanja
- Preverite, da metode pravilno uporabljajo this
- Uporabite TypeScript interface za definicijo tipa objekta
