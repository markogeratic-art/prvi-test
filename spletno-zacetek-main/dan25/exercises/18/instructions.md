# Vaja 18: Manipulacija JSON

## Cilji
- Spoznati JSON metode
- Pretvarjati objekte v JSON
- Shranjevati in pridobivati podatke
- Obvladovati JSON strukturo

## Navodila
1. Uporabite JSON metode:
   - JSON.stringify() za pretvorbo v niz
   - JSON.parse() za pretvorbo nazaj v objekt
   - Obvladovanje napak pri parsanju

2. Shranite objekte v lokalno shrambo:
   - Ustvarite objekt
   - Pretvorite v JSON niz
   - Shranite z localStorage.setItem()

3. Pridobite in pretvorite nazaj:
   - Pridobite z localStorage.getItem()
   - Pretvorite z JSON.parse()
   - Obdelajte prazne ali neveljavne podatke

4. Delajte z več objekti:
   - Shranite polje objektov
   - Dodajajte in odstranjajte objekte
   - Posodabljajte obstoječe objekte

## Pričakovani rezultat v script.ts
- Primeri JSON.stringify() in JSON.parse()
- Shranjevanje objektov v localStorage
- Pridobivanje in pretvarjanje nazaj
- Uporaba več objektov
- TypeScript interface za JSON objekte

## Namigi
- Preverite, da delujejo obe pretvorbi
- Poskusite shraniti različne strukture
- Preverite, da je obvladovanje napak učinkovito
- Uporabite TypeScript interface za tipiranje JSON podatkov
