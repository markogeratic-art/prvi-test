# Vaja 04: Pogoji

## Cilji
- Spoznati if, else if, else stavke
- Uporabiti logične operatore
- Primerjati nize in številke
- Uporabiti prompt() za vnos

## Navodila
1. Uporabite if, else if, else stavke:
   - Osnovni if stavek
   - Več pogojnih stavkov z else if
   - Zadnji else stavek

2. Uporabite logične operatore:
   - In (&&)
   - Ali (||)
   - Ne (!)

3. Primerjajte različne tipi podatkov:
   - Številke z >, <, ===
   - Nize z ===
   - Logične vrednosti

4. Uporabite prompt() za preprost vnos:
   - Pridobite vnos od uporabnika
   - Pretvorite vnos v ustrezni tip
   - Uporabite vnos v pogojih

## Pričakovani rezultat v script.ts
- Primeri if, else if, else stavkov
- Uporaba logičnih operatorjev
- Primeri primerjanja različnih tipov
- Uporaba prompt() za vnos
- TypeScript tipi za spremenljivke

## Namigi
- Preverite, da so pogoji pravilno napisani
- Poskusite kombinirati več pogojev
- Preverite, da so vrednosti pravilno pretvorjene
- Uporabite TypeScript tip `number` in `boolean`
