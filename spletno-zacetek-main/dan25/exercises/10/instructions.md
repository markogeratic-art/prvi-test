# Vaja 10: Osnovni izbor DOM

## Cilji
- Spoznati osnovne DOM metode
- Uporabiti getElementById
- Spremeniti vsebino in stil
- Dodajati in odstranjevati razrede

## Navodila
1. Uporabite getElementById:
   - Pridobite element z id-jem
   - Shranite v spremenljivko
   - Preverite, da je element najden

2. Spremenite vsebino:
   - Uporabite textContent za besedilo
   - Uporabite innerHTML za HTML
   - Preverite, da so spremembe vidne

3. Spremenite stil:
   - Uporabite style.property
   - Spremenite barvo ozadja
   - Spremenite velikost pisave

4. Dodajajte in odstranjevajte razrede:
   - Uporabite classList.add()
   - Uporabite classList.remove()
   - Uporabite classList.toggle()

## Pričakovani rezultat v script.ts
- Uporaba getElementById
- Spremembe vsebine
- Spremembe stilov
- Dodajanje in odstranjevanje razredov
- TypeScript type assertions za DOM elemente

## Namigi
- Preverite, da so elementi pravilno izbrani
- Poskusite kombinirati različne spremembe
- Preverite, da so spremembe vidne v brskalniku
- Uporabite TypeScript type assertions: `as HTMLParagraphElement`, `as HTMLInputElement`
