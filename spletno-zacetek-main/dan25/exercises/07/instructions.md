# Vaja 07: Metode polj

## Cilji
- Spoznati napredne metode polj
- Uporabiti forEach, map, filter, find
- Razumeti razliko med metodami
- Delati s transformacijami podatkov

## Navodila
1. Uporabite `forEach()`:
   - Izvedite akcijo za vsak element
   - Ne vrača novega polja
   - Uporabite za enostavno iteracijo

2. Uporabite `map()`:
   - Ustvarite novo polje
   - Transformirajte vsak element
   - Vrača polje enake dolžine

3. Uporabite `filter()`:
   - Ustvarite polje z omejitvami
   - Vrača le elemente, ki ustrezajo pogoju
   - Uporabite za filtriranje podatkov

4. Uporabite `find()`:
   - Poiščite prvi element, ki ustreza pogoju
   - Vrača en element ali undefined
   - Uporabite za iskanje specifičnega elementa

## Pričakovani rezultat v script.ts
- Primeri vseh štirih metod
- Primeri različnih uporab
- Primeri transformacij podatkov
- Primeri iskanja elementov
- TypeScript tipi za polja in povratne vrednosti
