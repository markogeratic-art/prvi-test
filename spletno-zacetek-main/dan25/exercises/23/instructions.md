# Vaja 23: Razredi

## Cilji
- Spoznati TypeScript razrede
- Uporabiti konstruktor
- Definirati metode
- Instancirati objekte
- Razumeti dedovanje

## Navodila
1. Uporabite razredno sintakso:
   - class deklaracija
   - constructor metoda
   - method definicije

2. Uporabite konstruktor:
   - Inicializacija lastnosti
   - Parametri konstruktorja
   - this referenca

3. Definirajte metode:
   - Instance metode
   - Static metode
   - Getter in setter metode

4. Instancirajte objekte:
   - new operator
   - Parametri pri instanciranju
   - Dostop do lastnosti in metod

5. Razumite dedovanje:
   - extends ključna beseda
   - super() klic
   - Prekrivanje metod

## Pričakovani rezultat v script.ts
- Razredna deklaracija s tipi
- Konstruktor z lastnostmi in tipi
- Metode razreda z povratnimi tipi
- Dedovanje z extends
- TypeScript tipi za vse lastnosti

## Namigi
- Uporabite TypeScript tipe za vse lastnosti
- Določite povratne tipe za metode
- Uporabite access modifiers (public, private, protected)
- Poskusite dodati getter in setter metode
- Preverite, da dedovanje pravilno deluje
